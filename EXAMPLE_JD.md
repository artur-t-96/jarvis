# Przykładowy Job Description - Senior React Developer

## O stanowisku
Poszukujemy doświadczonego Senior React Developera do zespołu rozwijającego platformę e-commerce.

## Wymagania (Must-have)
- Min. 5 lat doświadczenia komercyjnego w JavaScript
- Min. 3 lata pracy z React.js
- Bardzo dobra znajomość TypeScript
- Doświadczenie z Next.js
- Znajomość state management (Redux, Zustand, Context API)
- Praktyczna znajomość REST API i GraphQL
- Doświadczenie z Git (workflow, code review)
- Znajomość testowania (Jest, React Testing Library)
- Bardzo dobra znajomość HTML5, CSS3, responsive design
- Komunikatywna znajomość języka angielskiego (B2+)

## Wymagania dodatkowe (Nice-to-have)
- Doświadczenie z Node.js / Express
- Znajomość Docker i CI/CD
- Doświadczenie z AWS lub Google Cloud
- Znajomość metodyk Agile/Scrum
- Doświadczenie w projektach e-commerce
- Umiejętność optymalizacji performance aplikacji webowych

## Zakres obowiązków
- Rozwój i utrzymanie aplikacji webowych w React/Next.js
- Projektowanie i implementacja nowych funkcjonalności
- Code review i mentoring juniorów
- Współpraca z zespołem UX/UI przy implementacji designu
- Optymalizacja wydajności aplikacji
- Udział w planowaniu sprintów i estymacjach
- Dbanie o jakość kodu i dobre praktyki

## Oferujemy
- Umowa o pracę lub B2B
- Wynagrodzenie: 15,000 - 22,000 PLN netto (B2B)
- 100% praca zdalna lub hybrydowa
- Elastyczne godziny pracy
- Budżet szkoleniowy 5000 PLN/rok
- Prywatna opieka medyczna
- Karta MultiSport

## Proces rekrutacji
1. Rozmowa telefoniczna (30 min)
2. Zadanie techniczne (home assignment)
3. Rozmowa techniczna (90 min)
4. Rozmowa z managerem/CEO (60 min)
